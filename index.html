The Wicci System

The Wicci System is a collaborative content management system. It consists of twelve extensive components which have each been given their own git repository to allow them to be reused more flexibly. All components should be checked out and linked together appropriately before trying to build the Wicci System. The Wicci Project Page gives an overview and index.
